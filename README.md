# Pulse

SaaS analytics dashboard built as Project 1 of my 180-day frontend-to-product-engineer bootcamp.

## Stack

- **React 18** + **TypeScript** (Vite)
- **Tailwind CSS v4** for styling
- **Zustand** for client state _(coming soon)_
- **React Query** for server state _(coming soon)_
- **Recharts** for data visualization _(coming soon)_

## Getting Started

```bash
npm install
npm run dev
```

App runs at `http://localhost:5173`.
